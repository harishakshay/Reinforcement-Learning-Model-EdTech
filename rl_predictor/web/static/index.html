How the Simulation Works

State -> Action -> Reward -> Policy update, tuned for intentional discovery and reduced algorithm fatigue.

State Signals (10-D)

  • Novelty Score and Repeat Exposure
  • Creator Diversity and Topic Drift
  • Bubble Risk and Cognitive Load
  • Engagement Depth and Session Fatigue

Policy Loop

At each step, the agent reads user-context signals, picks one action (Diversify, Stay Balanced, Deepen Interest), then receives reward feedback to improve future recommendations.

Reward Philosophy

  • Aligned Recommendation: +1.0
  • Early Fatigue Prevention: +0.5
  • High-Confidence Alignment: +0.5
  • Misaligned Recommendation: -1.0
